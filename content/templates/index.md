# Templates

Шаблоны для копирования при новой сессии / новом баге / новом TC.

| Файл | Когда использовать |
|------|---------------------|
| [session.md](session.md) | При начале нового прогона — копировать в `sessions/YYYY-MM-DD-<тип>.md` |
| [bug.md](bug.md) | При находке нового бага — инструкция как занести в findings + сессию + zone |
| [test-case-template.md](test-case-template.md) | При новом формальном TC (legacy формат с backlog) |
| [bug-report-template.md](bug-report-template.md) | Bug-report для Google Docs (legacy формат для коллег) |
